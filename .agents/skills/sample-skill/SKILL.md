# サンプルスキル

このファイルは `.agents/skills/` フォルダの構造を示すためのサンプルです。

## 使用方法
1. このフォルダをコピーして新しいスキル名（例: `web-search`）に変更します。
2. `SKILL.md` の `name` と `description` を更新します。
3. そのスキルの具体的な指示やワークフローをこの下に記述してください。

## スキルの例
- プロジェクト固有のコーディング規約
- 特定のAPIの使用方法
- 複雑なリファクタリングの手順
