# Tepora ユーザーガイド

Teporaアプリへようこそ。
このガイドでは、Teporaのインストールから基本的な使い方、カスタマイズ方法までを分かりやすく解説します。

## 1. はじめに：Teporaとは？

Teporaは、ただのAIチャットツールではありません。あなたの会話や好みを記憶し、まるで専属のパートナーのように寄り添う「パートナーAI」です。

- **あなたを覚えている**: 昨日の会話も、あなたの仕事のプロジェクトも、Teporaは忘れません。
- **プライバシーファースト**: 全てのデータはあなたのPC内に保存され、外部に送信されることはありません。
- **Cyber Tea Salon**: 美しく、落ち着いたデザインで、至福の作業空間を提供します。

---

## 2. インストールとセットアップ

### システム要件

快適にご利用いただくために、以下のスペックを持つWindows PCを推奨します。

- **OS**: Windows 10 または Windows 11 (64bit)
- **CPU**: モダンなマルチコアCPU (Intel Core i5/i7/i9, AMD Ryzen 5/7/9 等)
- **メモリ (RAM)**: 16GB以上推奨 (8GBでも動作しますが、モデルによっては動作が遅くなる場合があります)
- **ストレージ**: 10GB以上の空き容量

### インストールの手順

1. 配布されたインストーラー (`tepora_x.x.x_x64-setup.exe`) をダブルクリックして実行します。
2. Windowsのセキュリティ警告（SmartScreen）が表示される場合がありますが、「詳細情報」をクリックし、「実行」を選択してください（※現在は証明書署名前のため）。
3. インストーラーの指示に従って、「次へ」または「インストール」をクリックします。
4. インストールが完了すると、デスクトップにTeporaのアイコンが作成されます。

### 初回起動とセットアップ

1. デスクトップのアイコンをダブルクリックしてTeporaを起動します。
2. 初回起動時は、AIモデルの準備などに数分かかる場合があります。「Cyber Tea Salonへようこそ」という画面が表示されるまでお待ちください。
3. **初期設定ウィザード**: 画面の指示に従って、使用する言語やAIモデル（エンジン）を選択してください。
   - **Engine**: 初心者の方はデフォルトのまま進めてください。
   - **Model**: 自動的に推奨モデルがダウンロードされます。

## 3. 基本的な使い方

### チャットを始める

メイン画面の下部にある入力エリアにメッセージを入力し、送信ボタン（紙飛行機アイコン）を押すか、Enterキーを押してください。
Teporaがあなたのメッセージに応答します。

### 記憶機能 (Memory)

Teporaとの会話の中で、重要な情報は自動的に「記憶」として保存されます。
特別な操作は必要ありません。次回以降、「この前の件だけど...」と話しかければ、文脈を理解して返答してくれます。

### 新しいセッション (New Session)

話の話題を大きく変えたいときは、画面左上の **[+] New Chat** (または新規作成アイコン) をクリックして、新しい会話を始めましょう。
過去の会話は左側のサイドバーからいつでも見返すことができます。

## 4. 設定とカスタマイズ

独自の「Tea Salon」を作り上げましょう。

### テーマの変更

設定画面（歯車アイコン）から、外観テーマを変更できます。

- **Tepora**: Cyber Tea Salonの世界観を体現した、デフォルトの美しいテーマ
- **Light**: 明るく清潔感のあるテーマ
- **Dark**: 目に優しく、没入感のあるテーマ
- **System**: Windowsの設定に合わせます

### 言語設定

設定画面の「一般 (General)」タブから、表示言語を切り替えることができます（日本語 / English / Español / 中文）。

### モデルの管理

カスタマイズ機能として、使用するAIモデルを変更することも可能です。

- 設定画面の「モデル (Models)」タブを開きます。
- **Character Model**: 会話に使用するメインのモデルを変更できます。
- **Professional Models**: 特定のタスク（コーディングなど）に使用する専門モデルを設定できます。
- **Model Management**: 新しいモデルのダウンロードや、Hugging Face形式のGGUFモデルの追加が行えます。

## 5. よくある質問 (FAQ)

**Q. インターネット接続は必要ですか？**
A. 基本的な会話は完全にオフライン（ローカル）で行われます。ただし、初回セットアップ時や新しいモデルをダウンロードする際にはインターネット接続が必要です。

**Q. 動作が重いです。**
A. 高度なAIモデルをPC内で動かしているため、相応のマシンスペックが必要です。他の重いアプリケーション（ゲームや動画編集ソフトなど）を終了してからTeporaを使用してみてください。

**Q. エラーが表示されます。**
A. アプリを再起動してみてください。それでも解決しない場合は、エラーメッセージをスクリーンショットに撮り、開発者またはサポートまでご連絡ください。
