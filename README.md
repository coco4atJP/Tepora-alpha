## ☕ Tepora（テポラ）へようこそ

Teporaは、ただの「便利なチャットボット」ではありません。
あなたのデスクトップにひっそりと佇む、サイバー・ティーサロン。そこには、あなたの言葉に耳を傾け、あなたを理解し、共に成長するパートナーが待っています。

ガラス細工のような透明感のあるインターフェースと、落ち着いた空間。
Teporaは、最先端のAI技術と「おもてなし」の心を融合させた、新しいデスクトップアプリケーションです。

## ✨ 特徴：あなたとの時間を大切にするために

### 1. 記憶という名の「絆」 (EM-LLM System)
「あれ、前にも話したよね？」——そんな体験を、AIとも。
Teporaは独自の記憶システム（EM-LLM）により、あなたとの会話から大切なエピソードを記憶します。嬉しいことがあった日も、少し疲れた日も、Teporaはこれまでの文脈を大切にしながら、あなたに寄り添います。

### 2. プライバシーを守る、安心の個室
Teporaは、あなたのPCの中で動作する「ローカルAI」です。
誰にも見られない、あなただけのプライベートな空間。会話の内容が外部に漏れることはありません。安心して、心の内をお話しください。

### 3. 個性豊かなパートナーたち
気分や目的に合わせて、対話する相手を選ぶことができます。

*   **Tepora (Main Agent)**:
    日常の何気ない会話から、ちょっとした相談まで。親しみやすく、あなたの心を解きほぐすムードメーカーです。
*   **Professional Agent**:
    調べ物や複雑なタスクはお任せを。豊富な知識とツールを駆使して、あなたの仕事をクールにサポートします。

## 🚀 はじめ方 (Getting Started)

Teporaとの生活を始めるのはとても簡単です。

### 推奨環境
- 少し強めのPC (GPU搭載を推奨)
- お気に入りの紅茶（必須ではありませんが、推奨です）

### インストール手順（一般ユーザー向け）

現在、Teporaは開発プレビュー版として提供されています。ご利用には以下の準備が必要です。

1.  **リポジトリの取得**:
    ```bash
    git clone https://github.com/coco4atJP/Tepora.git
    ```
2.  **セットアップ**:
    詳しくは [開発者ガイド](docs/guides/development.md) をご覧ください。

## 🛠️ エンジニア・開発者の皆様へ

Teporaは、最新の技術スタックで構築された野心的なオープンソースプロジェクトです。
「記憶を持つAI」「ローカルLLMの実用化」「Cyber Tea Salon UI」に興味がある方の参画を歓迎します。

*   **Architecture**: ステートレスなLLMサービス、LangGraphによるエージェント制御、TauriによるクロスプラットフォームGUI。
*   **Tech Stack**: Python 3.13, TypeScript (React), Rust (Tauri), LangChain/LangGraph.

詳細なドキュメントはこちら：
*   📂 [包括的アーキテクチャ仕様書](docs/architecture/ARCHITECTURE.md)
*   📂 [開発者セットアップガイド](docs/guides/development.md)

## 💐 謝辞 (Acknowledgments)

本プロジェクトは、多くの素晴らしいオープンソース技術と研究成果に支えられています。
